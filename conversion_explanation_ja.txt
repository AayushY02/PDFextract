============================================================
テキスト変換処理の仕様説明
============================================================

1. text_cleaner.py（テキスト正規化段階）

目的：
全項目共通で、文字表記を正規化する。

------------------------------------------------------------
A. 全角 → 半角への統一（英数字・記号）
------------------------------------------------------------

英数字および一般的な記号は、すべて半角に変換する。

例：
- ＡＢＣ１２３ → ABC123
- １２，３４５．６７ → 12,345.67
- （ ） → ( )

以下の記号は自動変換されないため、手動で半角に置換する。

〜  → ~
～  → ~
・  → ･
／  → /
＼  → \
＋  → +
－  → -
＝  → =
，  → ,
．  → .
；  → ;
：  → :
？  → ?
！  → !
｜  → |
＄  → $
％  → %
＠  → @
＾  → ^
＆  → &
＊  → *
＃  → #

------------------------------------------------------------
B. 全角スペース → 半角スペース
------------------------------------------------------------

全角スペースはすべて半角スペースに変換する。

例：
- 「工　事　名」 → 「工 事 名」

============================================================
2. dsl_engine.py（CSV書き出し直前）
============================================================

この段階では「工事名」項目のみ特別な変換を行う。
それ以外の項目には全角化処理は行わない。

------------------------------------------------------------
A. 工事名のみ全角に戻す
------------------------------------------------------------

以下を全角に変換する：
- 英数字
- 記号
- 半角カタカナ → 全角カタカナ
- 半角スペース → 全角スペース

例：
- ABC123 → ＡＢＣ１２３
- ｶﾀｶﾅ → カタカナ

------------------------------------------------------------
B. 工事名のみ二重括弧の番号を丸数字に変換
------------------------------------------------------------

工事名内の二重括弧で囲まれた番号（ポインタ）を
丸数字に変換する。

例：
- ((1)) → ①
- ((2)) → ②

補足：
- 括弧は半角・全角どちらも対象
  例：((3)) / （（3）） → ③
- ポインタとは、二重括弧で囲まれた番号を指す

------------------------------------------------------------
C. 工事名以外の項目
------------------------------------------------------------

工事名以外の項目は全角化しない。
半角スペースのみ削除する。

例：
- 「工 事 名」 → 「工事名」

============================================================
まとめ
============================================================

text_cleaner.py
- 全角英数字・記号を半角へ変換
- 全角スペースを半角スペースへ変換
- 定義済み記号を手動で半角に置換

dsl_engine.py（CSV直前）
- 工事名のみ全角へ戻す（英数字・記号・スペース・半角カタカナ）
- 工事名のみ ((1)) → ① に変換
- それ以外の項目は半角スペース削除のみ
